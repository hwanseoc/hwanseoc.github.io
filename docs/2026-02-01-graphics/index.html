Electron / CEF Apps

Web: Runs in user's browser. No installation, limited system access, smallest footprint.

PWA: Progressive Web App. Installable from browser, offline support, no app store, limited system access.

Electron: Chromium + Node.js bundled. Full system access, easy to develop, large bundle (~150MB+).

CEF: Chromium Embedded Framework. Just Blink rendering (no Node.js), more native control, smaller than Electron, harder to develop.

All use Chromium's Blink engine → same rendering as Chrome.
